# 📊 留学赛道小红书内容策略分析

> 基于真实运营经验的数据分析项目 | Python · pandas · matplotlib

---

## 项目背景

本人曾在一家留学机构担任运营兼职，负责小红书账号的内容发布与管理。在运营过程中发现账号互动数据表现不理想，为系统性找出原因并归纳改进方向，独立开展了本次数据分析项目。

**核心研究问题：在马来西亚留学赛道，哪类内容策略能带来更高的用户互动？**

---

## 数据说明

| 项目 | 详情 |
|------|------|
| 数据来源 | 小红书平台公开数据（手动采集）+ 自有账号后台数据 |
| 样本量 | 88篇笔记（自有账号25篇 + 3个竞品账号63篇） |
| 核心指标 | 综合互动量 = 点赞数 + 收藏数×2 + 评论数 |
| 分析工具 | Python（pandas、matplotlib） |

> 收藏加权×2，反映小红书算法对收藏行为的高权重。由于竞品账号阅读量不对外展示，采用综合互动量作为统一对比指标。

---

## 分析维度

- **内容类型**：干货攻略 vs 个人经历 vs 账号宣传
- **标题特征**：含/不含 Emoji、含/不含数字
- **账号对比**：自有账号 vs 3个竞品账号
- **爆文归因**：Top 20% 高互动笔记的共同特征

---

## 核心结论

### 1️⃣ 干货攻略互动量远超其他内容类型
干货攻略平均综合互动量为 **69.6**，是个人经历（20.2）的 **3.4倍**，是账号宣传（11.2）的 **6.2倍**。留学赛道用户的核心需求是获取实用信息。

### 2️⃣ 标题含 Emoji 可提升互动量 146%
含 Emoji 标题平均互动量 **95.9** vs 不含 **39.1**，差距达 **146%**，是影响最显著的单一变量。

### 3️⃣ 标题含数字可提升互动量 102%
含数字标题平均互动量 **94.5** vs 不含 **46.8**，提升幅度达 **102%**。数字能为读者建立明确的信息预期。

### 4️⃣ 爆文（Top 20%）核心特征
- 100% 为干货攻略类内容
- 61% 标题含数字
- 50% 标题含 Emoji

### 5️⃣ 竞品账号互动量是自有账号的 2.3 倍
表现最佳的竞品账号平均综合互动量高达 **175.3**，其内容几乎全为干货攻略且大量使用 Emoji 和数字，与上述结论高度吻合。

---

## 优化建议

基于数据结论，对留学类小红书账号提出3条可落地建议：

1. **聚焦干货攻略**：停止发布个人日常、账号宣传等低互动内容，选题聚焦申请材料清单、签证流程、生活费预算等高需求方向
2. **标题必须含数字**：将具体金额、时间、数量前置，例如将"生活费够吗"改为"马来西亚留学2000生活费够吗"
3. **标题加入1-2个 Emoji**：在标题头部或关键词旁加入相关 Emoji，避免堆砌，以1-2个为宜

> 以上建议已应用于账号实际运营策略。
